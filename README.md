# CLI Task tracker

Project idea - https://roadmap.sh/projects/task-tracker

## Возможности

- Сохранение задач со своим статусом, описанием и временем создания, обновления.
- Хранение задач в JSON файлах (папка data/).
- Обработка невалидных данных - удаление при обращении к ним с указанием причины.
- Автоматический инкремент идентификатора для задачи.
- Вывод списка задач с фильтрацией по статусу.
- Обработка логических ошибок - присваивание дубликатов описания или статуса для задачи.

## Установка из исходников

- `~/make build`

### Предварительные требования

- `go >= 1.25`
- Поддерживаемые ОС: Linux / macOS / Windows (пакет `storage` использует `filepath.Join`, код не зависит от платформы)
